# Apple Fall Game

A simple and fun game built with HTML, CSS, and JavaScript where you control a basket to catch falling apples.

## How to Play

1. Open `index.html` in your web browser.
2. Click the "Play" button to start the game.
3. Use your mouse to move the basket horizontally across the bottom of the screen.
4. Catch the falling apples to score points.
5. Miss three apples, or let 50 apples fall, and the game is over.
6. The game gets faster as you score higher.


## Features
S
* **Dynamic Apple Speed:** The speed of falling apples increases as you progress, providing a challenging experience. 
* **Responsive Design:**  Adapts to different screen sizes for optimal gameplay on various devices.
* **Sound Effects:** Includes audio feedback for catching and missing apples.
* **Game Controls:**
    * **Start:** Starts the game.
    * **Restart:** Resets the game to the beginning.
    * **Stop:** Stops the current game.
* **Game Over Conditions:**
    * Missing 3 apples.
    * 50 apples have fallen.
* **Score Display:** Shows your current score.
* **Game Status Messages:** Provides feedback on game start, reset, and game over.

## File Structure

* **`index.html`:** The main HTML file for the game's welcome screen.
* **`game.html`:** The HTML file containing the game interface.
* **`style.css`:** The CSS file for styling the game.
* **`index.js`:** The JavaScript file containing the game logic.
* **`catch.mp3`:** Sound effect for catching an apple (place your sound file here).
* **`uncatch.mp3`:** Sound effect for missing an apple (place your sound file here).

## Setup Instructions

1. Clone this repository to your local machine or download the files directly.
2. Make sure you have the `catch.mp3` and `uncatch.mp3` files in the same directory as the other files or update the paths in the JavaScript code accordingly.
3. Open `index.html` in a web browser to start playing.
